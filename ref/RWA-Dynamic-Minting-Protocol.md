# Origin Forge Protocol

## 源铸协议 — 一种革命性的代币发行范式

---

## 🎯 核心理念

**我们不是交易所，我们是源铸炉。**

传统的代币发行要么是固定价格 ICO，要么是 AMM 流动性池。前者无法反映真实需求，后者需要大量初始流动性。

**Origin Forge** 提出了第三种方式：**在源头锻造，让市场定价**。

---

## 🔥 什么是 Origin Forge？

### 源头锻造，价值内生

```
传统模式：固定供应 → 市场交易 → 价格发现
Origin Forge：市场需求 → 源头锻造 → 价值内生
```

Origin Forge 是代币的**诞生之地**。每一枚代币都在这里被锻造出来，价格由算法根据当前供需状态实时计算。

### 两种操作

| 操作 | 说明 |
|------|------|
| **锻造 (Forge)** | 投入稳定币，按当前价格锻造新代币 |
| **熔回 (Melt)** | 熔回代币，按当前价格换回稳定币 |

**你获得的代币是在炉中新锻造的，不是从别人手里买的。**
**你熔回的代币会化为原料，释放的价值来自协议储备。**

---

## 📈 价格是如何锻造的？

### 动态定价曲线

价格不是人为设定的，而是由锻造公式根据以下因素实时计算：

- **当前供应量** - 已锻造的代币数量
- **储备金额** - 炉中锁定的稳定币
- **市场活跃度** - 锻造与熔回的博弈结果

```
价格 = f(供应量, 储备金, 市场参数)
```

**锻造的人多 → 供应增加 → 价格上涨**
**熔回的人多 → 供应减少 → 价格下降**

这是真正的市场价格发现，没有做市商，没有人为干预。

---

## 🛡️ 炉火淬炼：地板价机制

### 每一次锻造都在淬炼价值底线

Origin Forge 的独特之处在于：**锻造活动本身会提升代币的价值下限**。

当市场活跃度达到一定阈值时，协议会自动：
1. 调整锻造参数
2. 提升地板价（最低熔回价格）
3. 永久锁定已获得的价值增益

**地板价只升不降** —— 炉火越旺，底线越高。

### 为什么这很重要？

```
传统代币：价格可以归零，项目方可以跑路
Origin Forge：地板价锁定，协议永远有能力按地板价回收
```

即使市场情绪低迷，你的代币也有一个数学保证的最低价值。

## 🔢 锻造参数

### 核心参数说明

| 参数 | 含义 | 作用 |
|------|------|------|
| `K` | 曲线斜率 | 控制价格对供应变化的敏感度 |
| `Target` | 目标资金比率 | 地板价计算的基准参数 |
| `RaiseStep` | 淬炼步长 | 每次地板价提升的幅度 |

### 价格公式

```
当前价格 c = W / (T - p)
地板价 f = W / (T + k)
```

其中：
- `W` = 炉中储备金总额
- `T` = 代币总供应量  
- `p` = 枢轴点（由 target 计算得出）
- `k` = 曲线参数

2. 早期用户锻造代币，价格从低位开始

4. 熔回代币可赎回对应资产价值

1. 社区点燃公平的 Origin Forge
2. 无预挖、无私募、无特权
3. 所有人通过锻造获得代币
4. 代币价值随社区发展自然增长
