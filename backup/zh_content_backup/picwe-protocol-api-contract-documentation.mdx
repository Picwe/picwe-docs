## 开发者注意事项

- 所有操作都是去中心化的，通过智能合约执行。
- $WEUSD 可以通过跨链合约铸造、赎回或跨链转移。
- 完全兼容 Web3 钱包，包括 MetaMask、OKX Wallet 和 Nightly。
- 有关详细的 API 集成，请参阅 [投资 API](/zh/investment-api)、[IRO/GLA API](/zh/iro-gla-api) 和 [市场 API](/zh/market-api) 文档。
