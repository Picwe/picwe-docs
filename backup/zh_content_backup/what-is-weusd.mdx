# 什么是 $WEUSD？

<div style={{ color: 'rgb(110,110,100)', clear: 'both', fontSize: '1.125rem', marginTop: '1.25rem' }}>

**$WEUSD** 是 PicWe 的**原生跨链稳定币**，为 PicWe 多链基础设施中的**稳定性、安全性和无缝交易**而设计。它作为整个生态系统的**结算层**，支持 **DeFi 操作、RWA 平台和跨链支付**。

---

### $WEUSD 的核心特性

| 特性 | 描述 |
|------|------|
| **跨链结算** | 在 PicWe 支持的网络（以太坊、BNB Chain、Arbitrum、Base、Movement 等）上原生运行。无风险第三方桥接，无托管方，无延迟。为需要统一流动性的 DeFi 协议和跨链应用设计。 |
| **价格稳定** | 与 USDC 1:1 锚定，始终可以 $1 USDC 赎回。用户可以通过存入 USDC 铸造 $WEUSD，或在任何支持的链上即时赎回。免疫算法稳定币或抵押不足稳定币面临的脱锚风险。 |
| **去中心化** | 完全由智能合约控制，而非中心化机构。流通中的每一个 $WEUSD 都由用户提供的 USDC 100% 抵押。无许可、可验证、完全透明的链上操作。 |

---

### 为什么 $WEUSD 很重要

$WEUSD 不仅仅是另一个稳定币——它是真实世界资产交易跨链流动性的基础。通过结合去中心化、安全性和无缝跨链结算，它使以下场景成为可能：

- **DeFi 平台**：获取稳定、无边界的交易和借贷流动性
- **RWA 项目**：跨多链安全结算投资
- **跨链应用**：无桥接风险执行支付和转账
- **机构**：以最小摩擦全球交易和转移稳定价值

### $WEUSD 运作原理

#### 铸造流程

1. 用户在任何支持的链上向铸造合约存入 USDC
2. 智能合约验证存款
3. 等值 $WEUSD 铸造到用户钱包
4. USDC 保持为 1:1 抵押品

#### 赎回流程

1. 用户向赎回合约提交 $WEUSD
2. 智能合约销毁 $WEUSD
3. 等值 USDC 释放给用户
4. 在任何支持的链上可用

#### 跨链转账

1. 用户在源链发起转账
2. $WEUSD 在源链锁定/销毁
3. Omni-Layer 协议中继交易
4. 等值 $WEUSD 在目标链铸造/释放
5. 结算在数秒内完成

### 安全模型

**完全抵押**

每个 $WEUSD 代币都由经审计智能合约持有的 USDC 1:1 支持。抵押率可随时在链上验证。

**无算法风险**

与算法稳定币不同，$WEUSD 不依赖可能在压力下失败的复杂机制。简单的抵押模型确保稳定性。

**去中心化托管**

没有中心化方持有支持资产。智能合约以透明、可审计的逻辑管理所有托管操作。

**自平衡机制**

Omni-Layer 协议的套利系统自动维护跨链价格平价，无需中心化干预。

### 使用场景

$WEUSD 是平台所有 RWA 交易的主要结算货币。投资者使用 $WEUSD：
- 参与 IRO（首次 RWA 发行）
- 在二级市场购买 RWA 代币
- 接收收益和分配

**跨链支付**

无传统桥接风险跨支持区块链发送价值：
- 快速结算（秒级，非分钟级）
- 低交易成本
- 无桥接运营商对手方风险

**DeFi 集成**

$WEUSD 设计为与 DeFi 协议无缝集成：
- 借贷抵押品
- 流动性提供
- 收益农场策略
