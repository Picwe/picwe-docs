# Omni-Layer 协议

<div style={{ color: 'rgb(110,110,100)', clear: 'both', fontSize: '1.125rem', marginTop: '1.25rem' }}>RWA 结算的跨链基础设施</div>

Omni-Layer 协议是 PicWe 的跨链基础设施，实现多个区块链网络间的无缝资产移动和结算。它为 $WEUSD 的多链功能提供基础，并支持整个生态系统的 RWA 交易。

---

## 协议概览

<Image src="/omni-protocol-investor-overview.svg" alt="Omni-Layer 协议架构" width={1200} height={500} style={{ marginTop: '1.25rem', marginBottom: '1.25rem' }} />

Omni-Layer 协议使用户能够使用任何支持链上的任何代币购买 RWA 资产，通过算法套利平衡确保最优执行。

## 交易流程

### 示例：跨链 RWA 购买

用户想使用 Arbitrum 上的 ETH 购买以太坊主网的 RWA 代币。

**步骤 1：意图提交**

用户在源链（Arbitrum）提交购买意图。

**步骤 2：路径解析与优化**

Omni 中继器：
1. 解析意图（源/目标识别）
2. 查询价格预言机获取当前汇率
3. 计算最优执行路径
4. 评估风险和滑点

**步骤 3：跨链执行**

```
┌──────────────────────────────────────────────────────────────┐
│                   ARBITRUM 执行                               │
├──────────────────────────────────────────────────────────────┤
│  用户 ETH ──▶ DEX 聚合器 ──▶ WeUSD 池 (Arbitrum)             │
└──────────────────────────────────────────────────────────────┘
                         │
                         ▼ Omni-Layer 桥
┌──────────────────────────────────────────────────────────────┐
│                 以太坊主网执行                                 │
├──────────────────────────────────────────────────────────────┤
│  WeUSD 池 (Ethereum) ──▶ RWA 市场 ──▶ RWA 代币转账           │
└──────────────────────────────────────────────────────────────┘
```

**步骤 4：资产交付**

用户在数秒内收到目标链上的 RWA 代币。

## WeUSD 自平衡机制

协议通过套利驱动的平衡维护跨链价格稳定。

### 运作原理

当链间出现价格差异时：

1. **机会检测**：系统监控跨池价格差异
2. **Keeper 执行**：Keeper 竞争执行有利可图的套利
3. **自然再平衡**：套利活动将价格推回锚定
4. **无中心化干预**：市场驱动机制确保稳定

1. **经济安全**：博弈论激励使参与者行为一致
2. **密码学证明**：跨链消息的数学验证
3. **自平衡池**：通过套利自动风险缓解
4. **无单点故障**：无中心化信任的分布式架构
