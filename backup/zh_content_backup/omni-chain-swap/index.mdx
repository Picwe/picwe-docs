# 全链兑换

PicWe 的全链兑换使用链抽象交易模型（CATM）让用户使用一条链上的资产购买另一条链上的资产，无需复杂的桥接或多次授权。这为 Web3 生态系统创造了更安全、更快速、更具成本效益的跨链流动性。

---

#### 支持的跨链功能

- **Movement → EVM（已上线）**\
  使用 Movement 链资产直接购买 EVM 链资产——无需中介。
- **Movement → Solana（即将推出）**\
  在 Solana 上启用原生资产交互，提供更多跨链选项。
- **EVM → EVM（即将推出）**\
  EVM 兼容链之间的平滑资产转移，无需重复授权。
- **EVM → Solana（即将推出）**\
  连接以太坊和 Solana 生态系统，实现无缝跨架构交互。
- **全链 → BTC（即将推出）**\
  直接从 Movement、EVM 和 Solana 链向比特币转移资产，开辟新的流动性途径。

---

通过 CATM，PicWe 正在解锁真正的全链互操作性，打破区块链之间的壁垒，为 DeFi、RWA 和 $WEUSD 用户实现高效的资产流动性。
