# CATM 模型

<div style={{ color: 'rgb(110,110,100)', clear: 'both', fontSize: '1.125rem', marginTop: '1.25rem' }}>PicWe 全链流动性的核心</div>

**链抽象交易模型（CATM）** 是 PicWe 全链资产流动性的支柱，可实现安全、无缝的跨链交易，无需依赖传统桥接。可以把它想象成数字资产的全球港口集装箱系统：您可以交易、持有或转移所有权，而无需在链之间物理移动资产。

#### CATM 的工作原理

1. **在原始链上发起订单**

- 用户在其当前区块链上下达交易订单。
- 订单详情通过中继器发送到目标链。
- 目标链上的代理为订单提供代币并接收质押资金。

2. **无需桥接直接购买资产**

- 直接在另一条链上交易任何资产，无需通过桥接转移代币。
- 资产在智能合约中锁定，直到交易完成。
- 消除与传统跨链桥接相关的风险。

3. **成本和效率节省**

- 消除两步转账的需要，降低交易和 gas 费用。
- 简化流程，实现更快、更经济的交易。

4. **智能合约控制**

- 所有操作，包括下单、结算和购买，都由智能合约处理。
- 用户对其资产保持完全控制。
- 任何人都可以使用自己的或第三方中继器参与。

---

#### 日常用语中的 CATM

想象一个全球集装箱港口网络：

- 您可以在任何港口购买集装箱中的货物并获得所有权证书。
- 如果价值上涨可以出售证书，或支付运费取回货物。
- CATM 对数字资产的工作方式类似，让您无需跨链移动资产即可交易和管理所有权。

---

**为什么 CATM 重要**

CATM 通过以下方式革新全链流动性：

- 降低跨链交易的成本并提高速度。
- 将资产安全锁定在智能合约中。
- 为跨区块链交易提供完全去中心化、无许可的框架。

通过 CATM，PicWe 实现真正的 Web3 互操作性，允许资产在链之间自由流动，同时让用户保持完全控制。
