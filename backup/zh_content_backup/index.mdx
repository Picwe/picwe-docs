# PicWe 入门指南

### 什么是 PicWe？

![picwe-starter-guide](/picwe-starter-guide.png)

PicWe 是全球首个全栈式 RWA（真实世界资产）资本市场生态系统，提供从资产代币化、发行到交易的端到端基础设施。平台结合原生稳定币（$WEUSD）、去中心化 RWA 市场和跨链结算能力，在传统金融与去中心化金融之间架起无缝桥梁。

### 愿景

PicWe 帮助机构、企业和个人通过代币化真实世界资产获取全球流动性。通过连接数字资产与现实世界市场，平台提供更快速、更透明、更便捷的方式参与数字金融和传统金融。

---

### 核心产品

**1. $WEUSD：结算稳定币**

$WEUSD 是 PicWe 的原生稳定币，专为 RWA 交易的跨链结算而设计。与传统稳定币不同，$WEUSD 作为**结算层**，实现多链统一流动性。

- 与 USDC 1:1 锚定，100% 抵押
- 原生跨链功能，无需第三方桥接
- 专为机构级 RWA 结算设计

**2. RWA 发行平台 (IRO)**

首次 RWA 发行（IRO）平台提供无许可的代币化真实世界资产发行框架：

- 基于智能合约的标准化发行流程
- 内置下行保护机制（国库支持的地板价）
- 在 PicWe DEX 即时开启二级市场交易
- 透明的抵押品管理和独立审计

**3. RWA 投资平台**

去中心化的代币化资产投资市场：

- 固定收益产品（债券、票据）
- 房地产投资
- 大宗商品和收益分享产品
- 风险分级投资选项

**4. PicWe Swap**

跨链资产兑换，支持在 PicWe 多链基础设施上无缝交易：

- 无需复杂桥接的直接兑换
- 社区驱动的流动性提供
- 低成本、高效执行

### 核心能力

| 功能 | 描述 |
|------|------|
| **RWA 发行** | 通过标准化智能合约在数分钟内发行代币化资产 |
| **跨链结算** | 在支持的区块链之间转移资产和结算交易 |
| **下行保护** | 由国库储备支持的协议强制地板价 |
| **全天候运营** | 完全自动化、去中心化运营，无需人工干预 |
| **透明定价** | 链上国库和供应数据提供可验证的资产估值 |

---

### 使用场景

- **资产发行方**：代币化真实世界资产，接入全球资本市场
- **投资者**：参与代币化债券、房地产和另类投资
- **机构**：使用机构级基础设施进行跨链交易结算
- **交易者**：在具有内置流动性的二级市场交易 RWA 代币

---

### 技术基础

PicWe 在多个区块链网络上运营，包括：

- BNB Chain (BSC)
- 以太坊
- Arbitrum
- Base
- Movement
- HashKey Chain
- Plume Chain

所有操作通过经审计的智能合约执行，确保安全、透明和去中心化。

### 开始使用

1. **连接钱包**：使用 MetaMask、OKX 钱包或兼容的 Web3 钱包
2. **获取 $WEUSD**：从 USDC 铸造或在支持的交易所购买
3. **浏览投资**：探索平台上可用的 RWA 产品
4. **投资或交易**：参与 IRO 或在二级市场交易
