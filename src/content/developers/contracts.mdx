## Notes

### Proxy Contracts

Investment module contracts are upgradeable proxies:
- Always use proxy addresses for interactions
- Implementation addresses may change during upgrades
- ABIs remain stable across upgrades

### Token Decimals

| Token | Decimals |
|-------|----------|
| WEUSD | 6 |
| USDC (BSC) | 18 |
| USDC (Base/Arb) | 6 |
| RWA Tokens | 18 |

### Verification

All contracts are verified on respective block explorers:
- BSCScan
- BaseScan
- Arbiscan
- HashKey Explorer
- Plume Explorer
