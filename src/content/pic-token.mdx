## Overview

**$PIC** is the native governance token of the PicWe ecosystem. It enables holders to participate in protocol governance, access premium features, and earn rewards from platform activity.

### 1. Governance

$PIC holders can:

- Influence platform development direction
- Propose changes to ecosystem parameters

### 2. Fee Discounts

Holding $PIC provides:
- Reduced Origin Forge fees
- Priority access to new features
- Enhanced platform benefits

### 3. Staking Rewards

Stake $PIC to:
- Earn protocol fee sharing
- Receive additional $PIC rewards
- Access premium investment opportunities

### 4. Ecosystem Access

$PIC enables:
- Early IRO access (whitelist)

## Governance Framework

### Proposal Types

| Type | Description | Quorum |
|------|-------------|--------|
| Parameter Change | Fee adjustments, limits | 10% |
| Feature Addition | New functionality | 15% |
| Treasury Spend | Fund allocation | 20% |
| Protocol Upgrade | Smart contract changes | 30% |

### Voting Process

1. **Proposal** — Submit with minimum $PIC stake
2. **Discussion** — Community review period (5 days)
3. **Voting** — Token-weighted voting (3 days)
4. **Execution** — Timelock then implementation (2 days)

### Voting Power

```
Voting Power = $PIC Balance × Time Multiplier
```

Longer holding = more voting power (up to 2.5x multiplier)
