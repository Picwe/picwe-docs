# PicWe Protocol API & Contract Documentation

PicWe uses a master-slave smart contract architecture deployed across multiple blockchains, including Move and EVM-compatible chains. These contracts power $WEUSD, cross-chain transfers, and mint/redeem operations.

Below are the current contract addresses for supported networks.

#### Notes for Developers

- All operations are decentralized and smart contract-based.
- $WEUSD can be minted, redeemed, or transferred across chains via the cross-chain contracts.
- Fully compatible with Web3 wallets like MetaMask, OKX, and Nightly.
